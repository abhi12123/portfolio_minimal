HushChat is a web chat application that works on websockets. Socket.io is used for websockets implementation. Backend is created using ExpressJs and hosted at Netlify. Any user can create a new room, or join an already existing room and start chatting with the members. Since there is no database created no information is stored anywhere. Data is transferred in real time between the room members

Stack

  • Front-end: ReactJs, W3.CSS
  • Back-end: ExpressJs
